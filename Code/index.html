Welcome to my calculator web application!






This small project is nothing particularly special, I just wanted to practice jQuery for my upcoming Client-side courswork. This calculator is far from perfect, it's even missing the decimal functionality, but it is not intended to be a perfect finished product. Feel free to make changes to the code, just make sure to follow the license guidelines.

I have replaced the eval method with the evaluate function from the "mathJS" library, it's much more secure and cannot be used to execute JS code as far as i'm aware.
